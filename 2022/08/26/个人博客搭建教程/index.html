发表于2022-08-26|更新于2022-11-15

环境配置

github

  1. 使用浏览器打开github首页 https://github.com/ ，点击右上角Sign up注册账户
  2. 根据提示输入相关信息创建个人账户即可
  3. 进入个人主页，先点击repositories再点击new创建仓库
  4. 仓库名字为username.github.io(username为你的用户名)，选择public，添加readme文件，最后点击最下方的create repositories按钮

git配置

  1. 安装git，官网：https://git-scm.com/
  2. 检查git是否安装成功
  3. 配置用户名和邮箱

node.js和npm

  1. 安装nodejs，官网 https://nodejs.org/en/
  2. 检查是否安装成功，安装node的同时会安装npm
