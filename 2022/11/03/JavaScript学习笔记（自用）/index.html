JavaScript学习笔记（自用）

发表于2022-11-03|更新于2022-11-15
|阅读量:

入门

  • js语句没有要求必须以分号结尾，可加可不加
  • 注释
  • alert()可以在浏览器弹窗显示内容
  • 在控制台打印日志

变量与数据类型

  • 使用var声明变量，使用“=”运算符赋值给变量
  • 变量命名规范：不能以数字开头，不能与关键字、保留字重复，采用驼峰命名法或者蛇形命名法
  • 六种数据类型
  • 字面量：也就是常量的意思

运算符

  • 自增运算符和自减运算符，一样有前缀和后缀之分
  • 等于运算符 == 与全等于运算符 === 的区别（!= 与 !== 同理）

条件语句

函数基础

  • 函数声明及调用
  • 函数的返回结果

对象

  • 对象是属性的无需集合
  • 对象的某个属性是函数时，则称该属性是对象的方法
  • this 关键字指向调用方法的对象
  • 对象的分类：自定义对象、内置对象、宿主对象、第三方库的对象

数组

  • 数组的定义
  • 遍历数组
  • 数组的常用方法

内置对象

  • Math 对象
  • Date 对象

正则表达式

  • 正则表达式可以匹配字符串，完成字符串的截取，按照规则进行替换，校验字符串内容
  • 正则表达式独立于编程语言，很多语言都支持正则表达式

ES2015基础语法

  • 使用 let 代替 var
  • 可以使用 const 定义常量
  • 当对象为常量时，可以修改其属性的值
  • 模板字符串
  • 解构赋值

函数进阶

  • 匿名函数：即没有名字的函数，一般用作回调函数
  • 立即执行函数：声明之后直接调用的函数，调用完成之后就找不到了
  • 闭包函数
  • 箭头函数

面向对象

  • 类
  • 继承

DOM基础

  • 获取节点
  • 事件类型
